# Desarrollo de ejercicios de Python – Tomás Fonseca

## Descripción

En este repositorio se encuentran almacenados todos los cuadernos de Python que contienen los ejercicios propuestos en clase por el profesor, desarrollados completamente y cumpliendo con todos los requisitos solicitados, incluyendo también el bono extra propuesto junto con los ejercicios.

Cada cuaderno contiene varios análisis lo más completos posibles, algunos aportes extra que he considerado oportuno incorporar y, al final de cada uno, están mis conclusiones sobre cada tema.
